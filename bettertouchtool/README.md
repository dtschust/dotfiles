# BetterTouchTool

`Default.bttpreset` is mostly the raw BetterTouchTool preset export. One stale
disabled automation action has been redacted so the repo does not store a local
path or home-automation credential.

Review the preset before importing because BetterTouchTool exports can include
stale disabled shortcuts, local paths, and machine-specific automation.

Import it from BetterTouchTool's preset manager.
